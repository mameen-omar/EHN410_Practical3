11 ## RSA Key Generation
12 
13 1. Open a Linux Terminal.
14 2. Navigate to the root directory containing the *rsakeygen* source code.
15 3. Run the *"make rsakeygen"* command.
16 4. An executable called *rsakeygen* will be created.
17 5. Use *"./rsakeygen"* to run the RSA Key Generation program (if no input parameters are specified, a help menu will be displayed)
18 6. A list of input parameters and respective default values can be seen below:
19 
20 | Parameter | Description | Default Value |
21 |:---------:|:--------------------------------------------------------------------------------:|:-------------:|
22 | -h | Prints out the help menu | |
23 | -b | Specifies the number of bits for the public/private keys to be generated | None |
24 | -KU | Specifies the file to which the **public** key will be written | None |
25 | -KR | Specifies the file to which the **private** key will be written | None |
26 | -key | Specifies the key for the initialisation of the RNG (in hex by default) | None |
27 | -kf | Specifies the path to the key for the initialisation of the RNG (hex by default) | None |
28 | -ascii | Specifies that the key used is in ascii instead of hex | Hex |
29 
30 ### RSA Key Generation Usage Example
31 
32 ```console
33 ./rsakeygen -b bits -KU public_key_file -KR private_key_file -key key
34 ```
